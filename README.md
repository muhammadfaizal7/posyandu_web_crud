# E-Posyandu Web CRUD

Aplikasi E-Posyandu berbasis web dengan Flutter dan Firebase untuk mendukung kegiatan posyandu secara digital.

## 🚀 Fitur Utama

- 📋 CRUD Data Balita, Ibu Hamil, dan Vaksinasi
- 📊 Grafik perkembangan anak dan status vaksinasi
- 📅 Jadwal dan Riwayat Vaksinasi
- 📁 Laporan Bulanan dengan Filter Bulan & Tahun
- 🧾 Ekspor Laporan ke PDF
- 🔒 Autentikasi menggunakan Firebase Auth

## 🛠 Teknologi yang Digunakan

- [Flutter](https://flutter.dev/)
- [Firebase Firestore](https://firebase.google.com/docs/firestore)
- [Firebase Auth](https://firebase.google.com/docs/auth)
- [PDF & Printing package](https://pub.dev/packages/pdf)
- [fl_chart](https://pub.dev/packages/fl_chart)
